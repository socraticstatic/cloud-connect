Approach

Restyle the existing AddGroupModal.tsx in place. No new components, no new routes. The data model, step logic, and save handler stay identical. Changes are CSS/layout only:

  • Modal container: 1056px max-width, 636px height, r=24
  • Dark overlay: rgba(27,27,29,0.56)
  • Stepper: horizontal numbered circles with connecting lines
  • Form inputs: h-9, r=8, border #686e74
  • Buttons: Cancel/Back (outline pill) + Continue/Create (filled pill), right-aligned
  • Remove header border, bg-fw-wash strips, icon
