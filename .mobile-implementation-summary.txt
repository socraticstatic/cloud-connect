USAGE
-----
The mobile views are automatically activated when the viewport width is < 768px.
No configuration needed - the app automatically detects device type and renders
the appropriate view.

To test mobile views in development:
1. Open browser DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M / Cmd+Shift+M)
3. Select a mobile device or set custom dimensions < 768px
4. Navigate to /manage, /connections/:id, or /groups
